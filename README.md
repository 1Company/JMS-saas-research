## Evaluation Criteria

1. **Market Size** - Is there enough demand?
2. **Competition** - Can we differentiate?
3. **Effort** - Can we build it quickly?
4. **Revenue** - Clear path to monetization?
5. **Synergy** - Does it leverage our skills/code?

# EventShare - Event Foto's Delen

## Status: LAUNCHED

## Samenvatting
EventShare is een platform waarmee eventbezoekers foto's delen via QR-code. Organisatoren scannen een QR-code bij het event, bezoekers uploaden foto's naar een gedeeld album, en na het event heeft de organisator alle foto's gecentraliseerd.

## Kernfeatures
- QR-code generatie per event
- Real-time foto uploads door bezoekers
- Gecentraal fotoalbum voor organisatoren
- Download alle foto's als ZIP
- Branding opties (logo, kleuren)
- Privé vs publieke albums
